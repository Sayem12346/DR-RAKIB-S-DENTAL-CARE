⚙️ BACKEND REQUIREMENTS (Node.js/Express + MongoDB)
═══════════════════════════════════════════════════════════════════════════════

Required API Endpoints:
• POST /api/appointments     - Store new appointment
• GET  /api/appointments     - Admin view all (filter by status/date)
• PUT  /api/appointments/:id - Update status (pending→confirmed→completed)
• POST /api/notices          - Create notice
• GET  /api/notices          - Public view (sorted, pinned first)
• PUT  /api/notices/:id      - Edit/pin notice
• DELETE /api/notices/:id    - Delete notice

Notification System:
• WhatsApp: Twilio API or WhatsApp Business API
• Email: SendGrid or Nodemailer
• SMS: Twilio fallback
